# Оновлення системи «Здорово — Облік робочого часу»
**Дата:** 27 травня 2025 року

---

## 🔐 Покращення входу через QR-код

Повністю перероблено інтерфейс сканування QR-коду для входу співробітників.

**Було:** стандартний некрасивий сканер з рамками в стилі 2000-х.  
**Стало:** сучасний повноекранний сканер з анімованою лазерною лінією, чистий дизайн без зайвих елементів. Виглядає як у сучасних банківських застосунках.

---

## 📊 Статистика для співробітника

Тепер кожен співробітник прямо на своїй сторінці бачить скільки він відпрацював — без необхідності звертатись до керівника.

Під кнопкою «Стоп» відображаються три картки:

| Картка | Що показує |
|---|---|
| **Вчора** | Загальний час попереднього робочого дня |
| **Тиждень** | Сумарно за поточний тиждень (пн–нд) |
| **Місяць** | Сумарно за поточний календарний місяць |

> Дані розраховуються за київським часом і оновлюються при кожному відкритті сторінки.

## 👤 Управління співробітниками

### Видалення співробітника
Адміністратор тепер може повністю видалити співробітника з системи. При видаленні автоматично видаляються всі його зміни та записи.

**Де:** Панель адміністратора → таблиця співробітників → кнопка «Видалити».

---

## 💰 Погодинна ставка та розрахунок зарплати

Додано індивідуальну погодинну ставку для кожного співробітника окремо.

- Кожному співробітнику можна встановити свою ставку (в гривнях за годину)
- Система автоматично розраховує **орієнтовну зарплату за поточний місяць**
- Зарплата = відпрацьовані години за місяць × погодинна ставка
- Нова колонка **«Зарплата»** у таблиці адміністратора відображає розраховану суму

**Де:** Панель адміністратора → таблиця → колонка «Зарплата» → клік на ставку (підкреслений текст з іконкою ✏️).

## ✏️ Повне управління змінами (нова функція)

Адміністратор отримав повний контроль над записами робочого часу.

### Редагування зміни
Якщо співробітник забув натиснути «Стоп» або вийшов не о тій годині — адміністратор може вручну відкоригувати час початку та кінця зміни.

**Де:** Таблиця → розкрити рядок співробітника → іконка ✏️ поруч із записом часу → змінити → «Зберегти».

### Додавання зміни вручну
Якщо співробітник зовсім забув відмітитись — адміністратор може створити зміну «з нуля».

**Де:** Таблиця → розкрити рядок співробітника → кнопка «➕ Додати зміну».

### Видалення зміни
Зайвий або помилковий запис можна повністю видалити.

**Де:** Таблиця → розкрити рядок → іконка ✏️ → кнопка «Видалити зміну» (червона).

## 📄 Експорт у Excel (CSV)

Додано можливість вивантаження всіх даних в Excel одним натисканням.

Файл містить наступні дані по кожному співробітнику:
| Стовпець | Опис |
|---|---|
| Співробітник | ПІБ |
| Логін | Логін у системі |
| Сьогодні (год) | Відпрацьовано сьогодні |
| Тиждень (год) | Відпрацьовано за тиждень |
| Місяць (год) | Відпрацьовано за місяць |
| Ставка (грн/год) | Погодинна ставка |
| Зарплата за місяць (грн) | Розрахована сума |

**Де:** Панель адміністратора → блок «Звіти по співробітниках» → кнопка «📄 Експорт в CSV».

> Файл відкривається напряму в Microsoft Excel або Google Таблицях. Числові значення розпізнаються коректно.

---

## 🌍 Стабільність та часовий пояс

Вся статистика (сьогодні / тиждень / місяць) розраховується за **київським часом (UTC+3)**, незалежно від того, де фізично знаходиться сервер. Це гарантує коректні звіти при роботі з хмарним хостингом.
